# 🌐 Portafolio Personal — Desafío Técnico Semana 2

Proyecto desarrollado como parte del **Bootcamp en Desarrollo Web** de BIT - Build Innovate Transform, correspondiente al Desafío Técnico de la Semana 2.

## 📋 Descripción

Página web de portafolio personal que muestra proyectos, habilidades técnicas e información de contacto. El proyecto aplica conocimientos de HTML semántico, CSS avanzado y publicación con GitHub Pages.

## 🛠️ Tecnologías utilizadas

- **HTML5** — Estructura semántica con etiquetas `<header>`, `<main>`, `<section>` y `<footer>`
- **CSS** — Flexbox, Grid, estilos responsivos y efectos hover
- **Git & GitHub** — Control de versiones
