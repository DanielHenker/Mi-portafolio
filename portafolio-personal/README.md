# Portafolio personal

Este proyecto es una pagina web sencilla de portafolio personal para Daniel Henker.
Daniel estudia Biologia en la Universidad Nacional y tambien participa en un bootcamp de desarrollo web.

Incluye:

- Presentacion personal.
- Tres proyectos de ejemplo, incluyendo una guia de plantas relacionada con Biologia.
- Lista de habilidades.
- Formulario de contacto.

Los textos de proyectos y datos de contacto se pueden cambiar despues.
